# FPS Overlay v1.7

## 快速开始

1. 运行 `fps_monitor.exe`
2. 启动游戏
3. FPS 显示在左上角

## 显示效果

```
120 FPS [V]   ← VSync 开启
120 FPS       ← VSync 关闭
```

**颜色**：🟢 ≥60 流畅 | 🟡 30-59 一般 | 🔴 <30 卡顿

## 操作

| 操作 | 方法 |
|------|------|
| 显示/隐藏 | 右键托盘 → Toggle |
| 退出 | 右键托盘 → Exit |

## 注意

- 先启动本程序，再启动游戏
- 退出程序不会关闭游戏
